## Technologies Used
- **EDA & OLAP**: pandas, Streamlit
- **Binning & Normalization**: pandas, Scikit-Learn
- **Statistical Testing**: SciPy (Chi-Square)
- **Association Rules**: MLxtend (Apriori, FP-Growth)
- **Dimensionality Reduction**: Scikit-Learn (PCA)
- **Machine Learning**: Scikit-Learn (Decision Tree, Naive Bayes, KNN, Linear Regression)
- **Visualizations**: Plotly

## Quick Start
1. Ensure you have Python installed.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the Streamlit application:
   ```bash
   streamlit run app.py
   ```
4. Once the app launches in your browser, upload the `tmdb_5000_movies.csv` file via the sidebar to begin analyzing!
